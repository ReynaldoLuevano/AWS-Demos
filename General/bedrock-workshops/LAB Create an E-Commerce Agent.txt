Tarea 3. Creando el Agente
Paso 9 - En el cuadro de búsqueda de la barra de navegación superior, escribe:bedrock. En los resultados de la búsqueda, en la sección Servicios, haz clic en Amazon Bedrock.
Paso 10. En el panel de navegación izquierdo, desplácese hacia abajo hasta "Build" y haga clic en "Agents".
Paso 11. Haga clic en "Create Agent"
Paso 12. En la ventana emergente coloque como nombre de agente "e-commerce-agent" y coloque la siguiente descripción "Product recommendations agent" y haz clic en "Create"
Paso 13. Aparece la vista "Agent Builder", donde configuraremos el modelo. 
Paso 14. En el apartado "Agent Resource Role", selecciona "Use an existing service Role" y del desplegable selecciona el unico valor disponible "workshop-Role"
Paso 15. Haz clic en Select  Model
Paso 16. En la ventana emergente desmarca el check "Bedrock Agents Optimized"
Paso 17. Selecciona Anthropic Claude Haiku. Asegurate que en el apartado Inference esta selecionado US Anthropic Claude Haiku 4.5 y haz clic en Apply
Paso 18. En el apartado Instructions for Agent, completa con lo siguiente:
"You are a Product recommendations agent for gift products, the user is trying to buy a gift for someone and you are trying to help identify the best products based on the filters in the action groups, ask questions to identify at least one of the input filters, gender, category or ocassion. Do not recommend any products that are not retrieved from the products API. Do not ask about the gender if it is obvious from the user input already. Always start by getting the full list of products from the API so you can know the proper filter values to be used in the API parameters. Always use a single value for each filter field, and adthere to the filtration values based on the first API call. And never tell the user about the API and its details"
Este  conjunto de instrucciones son importantes ya que guian al Agente en como comunicarse con el Usuario y en como utilizar las API
Paso 19. Haz clic en Additional Settings y en el apartado "User Input" selecciona la opcion Enabled
